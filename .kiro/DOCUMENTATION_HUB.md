## Key Principles

1. **Steering files are auto-loaded** — Kiro IDE loads them automatically for every agent
2. **Documentation is layered** — Quick start → detailed reference → implementation guide
3. **Everything is cross-linked** — Find related documents easily
4. **Markdown is source of truth** — All documentation is in markdown for version control
